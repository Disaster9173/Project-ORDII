В репозитории используется файл Sleep_health_and_lifestyle_dataset.csv
Он содержит сведения о сне,здоровье, образе жизни и их взаимосвязи.
Основной анализ выполнен в Проект_Султангеров.ipynb
-загрузка и первичный осмотр данных
-приведение типов и обработка пропусков
-расчёт базовых статистик(распределение рейтингов, количество отзывов, страниц)
-проверка корреляций между числовыми признаками
-построение нескольких графиков
-формулировка основных выводов по датасету
